Respostas perguntas comuns:

  • 1.Matriz linha:
  • Sempre possui apenas 1 linha.
  • 2.Matriz coluna:
  • Sempre possui apenas 1 coluna.
  • 3.Matriz nula:
  • Todos os elementos valendo 0.
  • 4.Matriz quadrada:
  • Número de linhas é igual ao número de colunas.
  • 5.Matriz transposta:
  • A transposta (t) de uma matriz A é uma matriz At com os mesmo elementos colocados em uma posição diferente. Se A(mxn) então At = (nxm).
  • 6.Matriz oposta:
  • Obtemos ao trocar/inverter os sinais dos elementos de uma matriz inicial.
  • 7.Matriz identidade:
  • É sempre do tipo quadrada, a diagonal principal só tem elementos 1 e os demais números são sempre 0.
  • 8.Matriz simétrica:
  • Quando fazemos a sua transposta e o resultado é igual à matriz original.
  • 9.Matriz inversa:
  • Quando duas matrizes quadradas se multiplicam e o resultado é uma matriz identidade.
  • 10.Matriz igual:
  • Duas matrizes são iguais quando são do mesmo tipo e possuem os mesmos elementos.
  • 11.Como somar as matrizes?
  • Somamos os elementos de mesma posição e colocamos o resultado final também na mesma posição.
  • 12.Como subtrair matrizes?
  • Primeiro você encontra a matriz oposta da segunda e depois realiza a soma entre elas.
  • 13.Quando é possivel multiplicar matrizes?
  • A multiplicação entre duas matrizes A e B só é possível quando o número de colunas de A for igual ao número de linhas de B. Isso porque devemos multiplicar cada uma das linhas por todas as colunas.
  • 14.Oque é a diagonal principal?
  • É a diagonal onde i=j, ex:
  • ! 0
  • 0 !
  • 15.Oque é a ordem de uma matriz:
  • Ordem de uma matriz refere-se ao seu número de linhas e colunas.
  • 16.Oque é uma matriz quadrada de ordem 2:
  • É uma matriz que contem 2 linhas e 2 colunas
  • 17.Oque é uma matriz?:
  • A matriz é uma maneira de representar dados dividindo-os em linhas e colunas. É representada por letras maiúsculas do alfabeto, como A, B, C, D. O índice que vem imediatamente após a letra diz a quantidade de linhas e de colunas que a matriz possui.
  • 18.Oque é uma matriz genérica?:
  • Ela indica o conjunto, as linhas e colunas como aij onde a representa o conjunto, i o número da linha e j o da coluna
  • 19.Como podemos obter a quantidade de elementos de uma matriz?:
  • Basta multiplicar o número de linha pelo número de colunas
  • 19.Ao somar a matriz A, com uma matriz neutra, que resultado teremos?:
  • O resultado será a Matriz A, pois temos a propriedade do elemento neutro na adição
  • 20.Oque é uma matriz I2?:
  • É uma matriz identidade, com a razão 2, ou seja, 2 linhas e 2 colunas
